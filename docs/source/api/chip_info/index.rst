Chip Information
================

This module provides utilities for parsing and handling chip identification information from the TROPIC01 secure element.

Available Classes
-----------------

.. autosummary::
   :nosignatures:

   tropicsquare.chip_id.ChipId
   tropicsquare.chip_id.serial_number.SerialNumber

Chip ID Structure
-----------------

The chip ID contains various information about the chip:

* Serial number (unique identifier)
* Hardware revision
* Firmware version
* Manufacturing data
